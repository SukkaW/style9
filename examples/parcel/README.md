# style9-parcel-example

Example using [style9](https://github.com/johanholmerin/style9) with Parcel.
See [.parcelrc](.parcelrc) & [.cssnanorc](.cssnanorc) for config and
[src/index.js](src/index.js) for usage.

## Installation

```sh
yarn
```

## Development

```sh
yarn start
```

## Build

```sh
yarn build
```
